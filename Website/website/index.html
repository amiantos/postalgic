What is Postalgic?

Postalgic is a static CMS for creating and publishing blogs. Write from iOS or the self-hosted web app, generate static HTML, and publish to your own hosting.

Your blog, your server, no platform in between. No algorithms, no ads, no surveillance—just your words on your corner of the web.

Key Features

  • Cross-Platform: Write from iOS or the self-hosted web interface
  • Static Output: Generate fast, secure static sites using Mustache templating
  • Publish Anywhere: Deploy to AWS S3, any SFTP server, or any Git repository
  • Own Your Content: Your data lives on your hosting, not someone else's platform

Get Started

Postalgic is free and open source. Check out the project on GitHub to get started.
